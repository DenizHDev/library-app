# Library App

## Description
This app makes it easier for users to keep track of books in their personal library. 
Users can see in real time if a book is available or checked out.

## Features
- User registration and login with secure password hashing
- Personal library — each user sees only their own books
- Add, checkout, return and remove books
- Flash messages for user feedback
- Responsive design with Bootstrap

## Technologies Used
- Python
- Flask
- SQLite
- Bootstrap
- HTML/CSS

## How to Run
1. Clone the repository: `git clone https://github.com/DenizHDev/library-app.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Visit `http://127.0.0.1:5000` in your browser
